Installing eLua

After building eLua, you need to install it (flash it to your board of choice) before using it. This section shows specific installation instructions for all the CPU/boards supported by eLua. Make sure that you already have a suitable eLua image for your target (either dowload one or build it yourself), then choose the platform of interest from the "Installing" submenu.

When building your eLua image, remember to specify "prog" to the scons command line in order to get a binary image file that's suitable for programming.
