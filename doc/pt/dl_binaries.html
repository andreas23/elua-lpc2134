Baixando imagens binárias pré-construídas de Elua

Se você tiver um hardware compatível com Elua, você não precisa ser um desenvolvedor profissional do mundo os sistemas embutidos e de entender os detalhes do processo de construção de Elua, para poder ter Elua em seus melhores kits.

O projeto eLua permite gerar imagens binárias pré-construídas para todas as plataformas suportadas.

Tudo que você tem a fazer é escolher o arquivo de imagem correspondente da tabela abaixo, grave-o em sua placa, conecte um terminal serial (ou Ethernet se sua placa suportar) e desfrute de Elua.

Os arquivos binários de eLua, tal como as distribuições de código-fonte, incluem alguns programas de exemplo em seu sistema de arquivos, assim você poderá executar e jogá-los (sim, você tem jogos também!:), seguindo as instruções na nossa página Usando eLua. Os programas de exemplo disponíveis, estão descritos na nossa página de exemplos
Se você precisar de um arquivo de imagem binário personalizado para uma plataforma já suporta (isto é: com um programa de execução automática (autorun), com algum código próprio no sistema de arquivos, com configurações de IP para sua rede local, .....) e você não sabe como gerar eLua, sinta-se a vontade para entrar em contato conosco explicano o que você precisa exetamente. Podemos (encontrar algum tempo para:) para construir um para você e, eventualmente, disponibilizá-lo aqui também.

Para entender como é composto um nome de arquivo em eLua (por exemplo elua_lualong_lm3s8962.bin), veja a nossa página Construindo eLua, ou pelo menos a última parte dele, onde o significado dos nomes dos arquivos provenientes do programa de geração do sistema é explicado.
