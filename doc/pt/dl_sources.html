Código Fonte

A última versão liberada é a v0.6 e está disponível para download aqui: ####
O pacote inclui o código fonte completo, documentação, scripts e exemplos de programa em Lua.

Depois de baixar e descompactar os arquivos para o seu computador, abra o documento index_pt.html na pasta /doc e você terá acesso a documentação. São os arquivos deste site quando da liberação da versão v0.6. Como este site é atualizado frequentemente, verifique sempre a versão online por novidades e atualizações de eLua.

Você encontrará instruções de como compilar eLua na página Montando eLua.



Repositório Público Subversion

Caso você prefira ter a última versão que ainda está em desenvolvimento ("bleeding edge"), simplesmente baixe-a em nosso repositório Subversion com o seguinte comando:

$ svn checkout svn://svn.berlios.de/elua/trunk

Para atualizar os seus arquivos use o seguinte comando:

$ svn update


Navegando pelo Repositório Subversion

Repositório Subversion para desenvolvedores

Se você deseja contribuir para o desenvolvimento de eLua e precisa salvar arquivos no repositório, siga os seguintes passos:

  • se você não tem uma conta no developer.berlios.de, crie uma antes de passar para o próximo passo.
  • entre em contato, especificando a sua identificação no BerliOS e nós iremos incluí-lo na lista de desenvovedores.

Faça o download do repositório:

$ export SVN_SSH='ssh -l '

$ svn checkout svn+ssh://svn.berlios.de/svnroot/repos/elua/trunk

Para atualizar os seus arquivos use o seguinte comando:

$ svn update
